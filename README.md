# Secure DevSecOps Pipeline

This project demonstrates a secure DevSecOps pipeline using Jenkins, Docker, Kubernetes, Helm, Argo CD, and AWS EC2.

## Tools Used
- AWS EC2
- GitHub
- Jenkins
- Docker
- Kubernetes
- Helm
- Argo CD
- SonarQube
- Trivy
- Prometheus
- Grafana

## Features
- CI/CD Automation
- Containerization
- Kubernetes Deployment
- Security Scanning
- Monitoring and Logging
